About This Exercise

This application utilizes The Open Movie Database (http://www.omdbapi.com/) to gather useful information about movies.

Notes:

- I decided not to filter out numbers and certain non-alphanumeric characters ( : , . ). movie titles look better this way. most other characters (~@}...) are filtered using an angular 'pipe'.

- a Movie 'id' represents it's current index on the 'movieList' array, and not it's 'imdbID' value from the database.

- All input validation is happening on the component, not the HTML template.

- I implemented some additional features inspired by the API abilities: 'Search' | 'Show More'.
